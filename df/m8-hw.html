married llama

From what I could tell, the main differences in Visual Studio is that you need to select the "x64" configuration and provide an entry point.

Select an x64 configuration for your project:
    Build->Configuration Manager->Active solution platform: x64
Project property changes:
    Properties->Linker->Advanced->Entry Point: main

Then you need realize that some MASM features are not supported or different in 64 bit.  For example, you need "end" at the end but cannot use "end main".  But the above settings should allow you to build example code from the book.
